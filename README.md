# Practical Cloud Infrastructure and Linux Systems Project

## Overview

This repository documents practical work with Linux systems, virtual machines, and major cloud platforms.  
The emphasis is on deploying, configuring, and managing resources using industry-standard cloud services and tools in real-world scenarios.

---

## Covered Areas

- Deployment and networking of Ubuntu on VirtualBox
- Linux command-line usage and Nginx web server configuration
- Snapshot management for virtual machines
- Deploying cloud-based virtual machines on Microsoft Azure
- Static website hosting using cloud services
- Windows Server setup and IIS configuration in the cloud
- Cloud storage configuration and disk resizing techniques
- Azure Blob Storage concepts and implementation
- Basic Docker containerization workflows
- Cloud cost estimation with AWS and Azure pricing calculators

---

## Technologies & Platforms

- Ubuntu Linux
- VirtualBox
- Microsoft Azure
- Amazon Web Services (AWS)
- Docker
- Nginx
- Windows Server (IIS)

---

## Use Case

This project illustrates core skills in cloud infrastructure management, Linux system administration, virtualization, and container-based workflows.  
The content is directly applicable to cloud operations and DevOps roles in real-world industry environments.
